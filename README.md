# Maxime Haegeman - Portfolio

A tech-minimalist portfolio site designed for a Data Scientist/Engineer. This project moves away from standard CMS bloat, opting for a raw, terminal-inspired aesthetic that reflects a focus on code, data, and efficiency.

**Live Site:** [www.maximehaegeman.dev](https://www.maximehaegeman.dev)

* **Terminal Aesthetic:** Monospaced typography and high-contrast "dark mode" design.
* **Pure Performance:** Built with vanilla HTML5 & CSS3. No frameworks, no build steps, zero dependencies.
* **Typewriter Effect:** Lightweight JavaScript implementation for the hero section.
* **Data-Object Layout:** Projects are presented as JSON-like data objects rather than standard cards.
* **Responsive:** Fully optimized for mobile and desktop viewports.
